# Statistikaameti veebiarenduse praktikakoha kodutöö

Kodutöö tegija on Mihkel Hain

## Projekti kasutamise juhend

Projekti saab alla laadida kahel viisil:
1. Kasutades git: `git clone <projekti-url>`
2. Laadides alla projekti `.zip` faili ja pakkides selle lahti.

Kui projekt on allalaaditud veendu, et sinu arvutis on olemas Node.js. Seejärel ava terminal projekti juurkaustas ja käivita:
### `npm install`

Projekti käivitamiseks brauseris kasuta käsku:
### `npm start`
Projekt peaks avanema aadressil "http://localhost:3000/"

## Playwright testid

Kui on vaja käivitada lisatud End to End (E2E) teste peab jooksutama:

### `npm playwright test`
